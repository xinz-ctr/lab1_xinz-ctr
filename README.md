THIS IS A STUDENT EVALUATOR AND ARCHIVER PROGRAM

WHAT THE PROGRAM DOES WHEN YOU LAUNCH THE grade-evaluator PYTHON FILE

The program will calculate the weighted grade for each assignment depending on their assigned weights, and then it will check if the total weight of the assightnments is 100, then it will calculate the grade out of 100 for both the summative and the formative, and then check if the student scored atleat 50 percent in both then it will print the status passed otherwise it will print failed, the program also calculates the GPA on a 5.0 scale, and finally checks the formative with the highest weight and if you failed it you are elligible for resubmission.

WHAT THE PROGRAM DOES WHEN YOU RUN THE organizer SHELL SCRIPT

It creates an archive directory if it does not exist, renames the grades.csv file by adding a timestamp and archives it into the archive file, then creates a new grades.csv file. It also log the archiving detail of the operation into a file named organizer.log
